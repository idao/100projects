为什么有多个 Opera 版本？

Opera 计算机版具有快速迭代的开发周期，这意味着大部分新功能需经历三个阶段。

首先，在早期试验阶段，会将功能引入 Opera developer 版，然后当功能更稳定时，将移至 Opera beta 版。最后，经过我们的完善，会将功能发布于 Opera 稳定版中，满足大部门用户的日常需求。您可以访问我们台式机团队的博客，了解 Opera beta 版和 Opera developer 版的最新增加功能。
